# Lógica de Programación 3 — Factorial en JavaScript

Este proyecto fue desarrollado en JavaScript como práctica de lógica de programación.

## Descripción

El programa solicita al usuario un número y calcula su factorial.

El sistema:

- Solicita un número mediante prompt.
- Valida que el dato ingresado sea de tipo numérico.
- Calcula el factorial del número ingresado.
- Muestra el resultado en consola y mediante alert.
- En caso de error, vuelve a solicitar un valor válido.

## Ejemplo

bash id="g4ud0f" Entrada: 5 Salida: 120 

bash id="9lr7pz" Entrada: 6 Salida: 720
